# API-Trading-Scraper
Problem we're looking to solve - Helping beginners understand the stock market/investing options better.
Aim - to increase financial literacy for anyone interested.
Approach - utilizing a web scraper to gather investing advice and send information to the website that will provide advice and insight into where that advice is coming from. Hoping to have beginner articles, investing/stock advice and a home page to introduce it all.
